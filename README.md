# Bitilda — Standalone Landing Page

A bold, editorial landing experience for Bitilda's borderless money app.

## Run locally

No build step is required. Open `index.html` directly, or serve the directory:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then visit `http://127.0.0.1:4173`.

## Structure

- `index.html` — page content and semantic structure
- `styles.css` — complete responsive visual system
- `animations.js` — reveal, counter, scroll, and pointer interactions
- `assets/` — Bitilda product and lifestyle artwork
